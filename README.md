# AI Assets Hub

Прототип веб-приложения для каталога AI-ассетов: карточки, детали, репозиторий, спецификация, отправка материалов и документация.

## Стек

- React 19, TypeScript
- Vite 6
- Tailwind CSS 4
- React Router 7
- Lucide React (иконки)

## Требования

- Node.js 20+ (рекомендуется LTS)

## Установка и запуск

```bash
npm install
npm run dev
```

Сборка и предпросмотр продакшен-сборки:

```bash
npm run build
npm run preview
```

## Маршруты

| Путь | Страница |
|------|----------|
| `/` | Каталог |
| `/assets/:id` | Карточка ассета |
| `/repository` | Репозиторий |
| `/specification` | Спецификация |
| `/submit` | Отправка |
| `/docs/:slug` | Документация |

## Структура

Исходный код находится в `src/`: страницы в `pages/`, компоненты в `components/`, данные и типы в `data/` и `types/`.
